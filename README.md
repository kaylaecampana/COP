# Carousel of Progress — Textual Analysis

A close reading of the *Walt Disney's Carousel of Progress* 2026 script, prepared
on **July 5, 2026** — the day the attraction's theater stopped turning for a
rewrite intended to move the whole show forward sixty years.

## Contents

- **`Progress Script 2026.txt`** / **`COP - Script 2026.pdf`** — the source script.
- **`carousel-progress-analysis.html`** — an interactive, visual version of the
  analysis. Open it in any browser (it is fully self-contained — no build step,
  no external assets).

## The interactive analysis

`carousel-progress-analysis.html` reframes the show as the rotating stage it is:

- A **circular stage selector** — click any act (I–IV) and a clock-hand pointer
  sweeps to it while a detail card shows that era's marvel, mishap, and a line
  of dialogue.
- A **"60-year update" toggle** that relabels every era live
  (1900s→1960s, 1920s→1980s, 1940s→2000s, mid-1990s→mid-2050s), dramatizing the
  analysis's central finding: the "modern day" act has been frozen since 1994.
- Supporting sections on the show's **uneven structure**, its **fixed family of
  characters**, the **running gags** that recur verbatim across the decades, and
  the **frozen-present problem** the closing-night rewrite inherits.

The page is theme-aware (light and dark) and responsive.
